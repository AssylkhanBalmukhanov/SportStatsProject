# SportStatsProject
# Overview
A comprehensive football analytics toolkit combining statistical modeling, machine learning, and data visualization to analyze event data and predict outcomes.

#Core Features
Match Prediction: Uses Poisson distribution and Monte Carlo simulations to predict match results based on team strength.

Expected Goals (xG): A classification model evaluating shot quality and scoring probability.

Expected Pass (xP): Predicts pass completion likelihood based on spatial difficulty and coordinates.

Player Clustering: Uses K-Means clustering to group players by statistical similarity and uncover tactical roles.

Team Passing Networks: Applies graph theory to visualize pass frequency, central hubs, and overall possession structure.

Computer Vision (Experimental): Basic player and ball tracking from broadcast footage using OpenCV.

## ML Features

Ability to apply common statistics to uploaded football videos.
